## Overview

Manually verify that the page renders correctly across three target viewports: mobile (375px), tablet (768px), and desktop (1280px). Checks cover image scaling, typography, and CTA button alignment.

---

## Test Environments

| Viewport | Label   | Width  |
|----------|---------|--------|
| Mobile   | Mobile  | 375px  |
| Tablet   | Tablet  | 768px  |
| Desktop  | Desktop | 1280px |

---

## Test Cases

### 1. Page Renders Without Layout Breakage

**Steps:**
1. Open the page in a browser.
2. Use DevTools (Chrome/Firefox) to set viewport to each target width.
3. Visually inspect the page at each width for overflow, clipping, or collapsed sections.

**Expected Result:**
- No horizontal scrollbar appears at any viewport.
- No elements overflow outside their containers.
- All sections are visible and structurally intact.

| Viewport | Pass / Fail | Notes |
|----------|-------------|-------|
| 375px    |             |       |
| 768px    |             |       |
| 1280px   |             |       |

---

### 2. Image Scaling

**Steps:**
1. At each viewport width, inspect all images on the page.
2. Verify images scale proportionally and do not stretch, distort, or overflow.
3. Confirm hero/banner images fill their containers correctly.

**Expected Result:**
- Images maintain aspect ratio at all viewports.
- No pixelation or distortion visible at 1x screen density.
- Images do not overflow their parent containers.

| Viewport | Pass / Fail | Notes |
|----------|-------------|-------|

---

### 3. Font Sizes

**Steps:**
1. At each viewport, inspect heading (h1–h3), body text, and caption/label elements.
2. Verify font sizes are readable and appropriately scaled for the viewport.
3. Check that no text is clipped, overlapping, or truncated unintentionally.

**Expected Result:**
- Headings are visually prominent and legible at all viewports.
- Body text is readable (minimum 14px equivalent) on mobile.
- No text overflows its container or overlaps adjacent elements.

### 4. CTA Button Alignment

**Steps:**
1. Locate all Call-to-Action (CTA) buttons on the page.
2. At each viewport, verify button alignment, sizing, and spacing.
3. Confirm buttons are fully tappable/clickable (minimum 44x44px touch target on mobile).

**Expected Result:**
- Buttons are centered or aligned per design spec at each viewport.
- On mobile (375px), buttons are full-width or properly centered with adequate padding.
- No button is partially hidden, clipped, or overlapping other elements.
- Sufficient spacing between stacked buttons on mobile.
