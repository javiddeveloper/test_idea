# Seiko 5 Landing Page — Frontend Spec

**Repo:** javiddeveloper/test_idea  
**Project:** ttttt  
**Type:** frontend / boilerplate  
**Stack:** Raw HTML + CSS (no frameworks, no JS)

---

## Overview

A single-page static HTML file showcasing the Seiko 5 watch. The page consists of one full-viewport hero section with a watch image, product name, tagline, and a CTA button.

---

## File Structure

```
index.html       ← single deliverable file (HTML + embedded or linked CSS)
assets/
  seiko5.jpg     ← watch product image
```

---

## Page Sections

### Hero Section

The only section on the page. Must occupy 100vh.

| Element        | Tag          | Content                              |
|----------------|--------------|--------------------------------------|
| Watch image    | `<img>`      | High-res product photo of Seiko 5    |
| Product name   | `<h1>`       | `Seiko 5`                            |
| Tagline        | `<p>`        | `Built to Last. Priced to Impress.`  |
| CTA button     | `<a>` / `<button>` | `Shop Now`                     |

---

## Acceptance Criteria

- [ ] Single `index.html` file renders without JS
- [ ] Hero section fills the full viewport height (`100vh`)
- [ ] Watch image is visible and responsive
- [ ] `<h1>` displays `Seiko 5`
- [ ] Tagline is present below the title
- [ ] CTA button links to `#` (placeholder) with hover state
- [ ] No external frameworks (Bootstrap, Tailwind, etc.)
- [ ] No `<script>` tags
- [ ] Page is responsive on mobile (320px+) and desktop
